机电工程师

北京 全职 硬件研发 期权激励
→

职位亮点

  • 参与人形机器人 0 到 1 研发
  • 核心团队岗位，决策空间大
  • 可参与整机架构设计与样机落地
  • 期权激励

岗位职责

  • 负责人形机器人整机机械架构设计，完成从概念设计到详细设计的全流程
  • 负责腿部连杆、关节连接件等核心结构设计，兼顾刚性、轻量化与可靠性
  • 进行关键受力件强度校核与仿真分析，确保跳跃、冲击等工况下结构安全
  • 输出 2D 工程图并对接加工厂，推进 CNC 加工、装配与问题闭环
  • 处理装配干涉、公差配合、表面处理等 DFM 问题
  • 设计内部线束走线方案，降低运动过程中线缆拉扯、磨损风险
  • 协同算法、嵌入式工程师完成整机调试与迭代

任职要求

  • 精通 SolidWorks / CATIA（至少一种），具备精密机械设计经验
  • 有足式机器人（四足/双足）机械设计经验，理解腿式机构动力学特性
  • 熟悉电机选型、减速器（谐波/行星）、轴承等传动部件
  • 能独立完成结构设计、出图、打样、装配问题定位
  • 有较强工程落地能力和跨团队协作能力

加分项

  • 有机器人竞赛/创业项目经验
  • 有量产导向设计经验

机器人决策算法工程师

北京 全职 算法研发
→

职位描述

负责人形机器人系统中的决策算法开发与优化。岗位聚焦群体智能与多机器人协同决策，参与策略设计、仿真验证、真机联调和系统迭代，推动算法在真实机器人平台上的落地应用。

岗位职责

  • 负责人形机器人系统中的决策算法开发，包括角色分配、行为决策、任务协同、路径规划及多机器人协作
  • 面向群体智能场景，设计并优化单机与多机协同决策逻辑
  • 搭建或使用仿真环境进行策略验证，并推动算法向真机部署
  • 与感知、控制、通信团队协同联调，提升系统的稳定性、实时性和整体表现
  • 基于测试数据、日志和视频分析，持续优化机器人在复杂场景下的决策能力

任职要求

  • 计算机、自动化、机器人、人工智能、控制等相关专业，本科及以上学历
  • 熟练掌握 Python 或 C++，具备良好的代码能力和工程实现能力
  • 熟悉以下一个或多个方向：群体智能、多智能体决策与协同、强化学习、路径规划、行为树/状态机、机器人控制与运动规划
  • 具备较强的问题分析能力和系统调试能力
  • 有机器人项目、机器人竞赛或相关研发经验者优先

加分项

  • 有 RoboCup、人形机器人、群体智能或博弈决策相关经验
  • 熟悉 ROS / ROS2 通信框架
  • 有强化学习训练、仿真平台开发或 Sim2Real 经验
  • 有多机器人协同或真实机器人落地经验

机器人视觉感知工程师

北京 全职 算法研发
→

职位描述

参与 RoboCup 人形机器人相关项目研发。岗位聚焦视觉感知算法的工程化落地，主要包括视觉代码 C++ 化、YOLO 目标识别，以及基于场地点线特征的定位与重定位。你将参与从算法开发、系统集成到真机联调的完整研发过程。

岗位职责

  • 负责 RoboCup 人形机器人视觉感知模块的开发、优化与部署
  • 推进现有视觉算法代码的 C++ 化重构与工程化实现，提升实时性、稳定性与可维护性
  • 基于 YOLO 或相关目标检测框架，实现比赛场景中的敌我目标识别
  • 基于球场点、线、圆弧等几何特征，实现机器人在场地中的定位与重定位
  • 与决策、控制、运动等模块协同联调，支撑机器人系统稳定运行
  • 针对真实环境中的光照变化、遮挡、运动模糊等问题进行鲁棒性优化
  • 基于测试数据、日志和视频持续迭代感知算法性能

任职要求

  • 计算机、自动化、机器人、电子信息、人工智能等相关专业，本科及以上学历
  • 熟练掌握 C++，具备良好的工程实现能力和代码规范意识
  • 熟悉计算机视觉、机器人视觉或目标检测相关方法
  • 有以下一个或多个方向经验者优先：YOLO / 目标检测模型训练、部署与优化；OpenCV 图像处理与几何视觉；点线特征提取、PnP、位姿估计、视觉重定位；机器人定位、环境感知或多传感器融合
  • 能独立完成算法调试、性能优化和模块联调
  • 具备较强的问题分析能力和团队协作能力

加分项

  • 有机器人竞赛、RoboCup 或具身智能相关项目经验
  • 有真实机器人视觉系统开发经验
  • 熟悉 ROS / ROS2 通信框架
  • 有 TensorRT、ONNX、模型部署加速经验
  • 有视觉 SLAM、重定位或几何约束优化经验

职位亮点

  • 参与 RoboCup 人形机器人核心研发项目
  • 面向真实机器人系统的视觉感知落地
  • 涵盖目标识别、场地定位、工程部署与真机联调
  • 与决策、控制、运动团队协同推进完整系统实现

岗位定位

岗位职责

  • 搭建系统化、阶梯式的具身智能技术启蒙学习体系
  • 围绕智能硬件、机器人控制、AI 感知与交互、简单编程实践等内容设计课程与活动
  • 引导学生从认知理解到动手实践，逐步建立科技思维与工程意识
  • 全面提升青少年科技素养、逻辑思维能力、动手实践能力与创新创造能力
  • 为学生未来参与科创竞赛、学科拓展及专业选择打下坚实基础

任职要求

  • 985/211 高校工科相关专业在校生或应届毕业生优先
  • 无需提前具备相关技术基础与教学经验，公司提供全方位体系化岗前培训
  • 培训内容涵盖教学方法、课程体系、具身智能基础知识、课堂组织与互动技巧等
  • 经统一培训并通过考核后即可正式上岗
  • 热爱教育事业，乐于分享知识与热爱，具备良好的表达与组织能力
